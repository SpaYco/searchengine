# SearchEngine

This is a realtime search engine

## What it does

- Lists articles
- You can search through the articles in realtime
- Every search will be saved and the top 100 will be listed

## Built With

- Ruby
- Ruby on Rails
- VSCode

## Deployment

You can check a live demo [HERE](https://thawing-ravine-16372.herokuapp.com/)

## Setup

Install gems with:

```console
bundle install
```

Setup database with:

```console
   rails db:create
   rails db:migrate
   rails db:seed
```

## Usage

Start server with:

```console
    rails server
```

Open `http://localhost:3000/` in your browser.

## Run tests

```console
    RSpec
```
